sync_gkeep_grocy 
appdaemon system packages : gcc g++
appdaemon python packages : Requests gkeepapi
module dependencies : GrocyApi
installation :
    1 : create a Google password app
    2 : create a Grocy key API (check in Grocy : Settings/Manage API keys)
    3 : set up your apps.yaml
    4 : copy sync_gkeep_grocy.py to your appdaemon apps directory
